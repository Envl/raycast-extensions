# Listen Changelog

## [1.0.0] - Initial Release

### Added

- Real-time speech transcription using macOS native Speech Recognition API
- Support for 15 languages (English, Spanish, French, German, Italian, Japanese, Korean, Chinese, Portuguese, Russian, Dutch)
- AI refinement with Raycast AI to polish grammar, punctuation, and formatting
- On-device recognition mode for privacy
- Auto-refine preference to automatically polish transcriptions when recording stops
- Live streaming with partial results appearing as you speak
- Manual stop control to end recording at any time
- Paste action to insert transcription directly into active application
- Copy action to save transcription to clipboard
- Word and character count display
